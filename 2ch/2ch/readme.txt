 2ch-mode

 by Masashi Hattori

■ 2ch-modeとは？

	xyzzy上で動くへなちょこ２ちゃんねるブラウザです。
	利用条件等に関しては、COPYING.txt をご覧下さい。

	２ちゃんねる <http://www.2ch.net/>

■ 動作環境

	xyzzy-0.2.1.184以降
	www-0.0.1.2 (2001/03/07)以降
	（動作にはwww-modeが必要です。）

■ インストール

    1. ファイルを展開する。

	"*.l" はsite-lisp/2ch/以下に置いて下さい。
	"dolib.dll" はxyzzy.exeと同じ場所に置いてください。

    2. バイトコンパイルする

	1. M-x load-library[Enter]
	2. Load library: 2ch/makefile[Enter]
	3. M-x 2ch-make-clean[Enter]
	4. M-x 2ch-make[Enter]

    3. 設定

	-- ~/.xyzzy (siteinit.l)

		(autoload '2ch "2ch/main" t)

	-- ~/.2ch/config.l

		設定ファイル。同梱のconfig.lをコピーして編集して下さい。

    4. xyzzyを再起動。

    5. 起動

	M-x 2ch[Enter]

	初回の起動時には板一覧を取りに行くので時間がかかります。
